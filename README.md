# Bank Loan Report Analysis (SQL)

This repository contains a comprehensive SQL-based analysis of a Bank Loan dataset. The project focuses on generating summary reports and analytical insights related to loan applications, funding, repayments, interest rates, and borrower risk using structured SQL queries.

## Project Overview

The project analyzes a bank loan database to evaluate overall loan performance, month-to-date trends, loan quality, and borrower characteristics. The SQL queries are designed for reporting and decision-making in the banking domain.

### Dataset Information
- Database Name: Bank Loan DB
- Table Name: bank_loan_data

Key columns used:
- id
- issue_date
- loan_amount
- total_payment
- loan_status
- int_rate
- dti
- term
- emp_length
- purpose
- home_ownership
- address_state

## Tools and Technologies Used

- SQL
- Relational Database
- Aggregate Functions (SUM, COUNT, AVG)
- Date Functions (MONTH, DATENAME)

## Analysis Sections

### A. Bank Loan Report | Summary

#### Loan Application Analysis
- Total loan applications
- Month-to-date (MTD) loan applications
- Previous month-to-date (PMTD) loan applications

#### Funded and Collected Amount Analysis
- Total funded amount
- MTD and PMTD funded amount
- Total amount collected
- MTD and PMTD amount collected

#### Interest Rate and Risk Metrics
- Average interest rate
- Average debt-to-income (DTI)
- MTD and PMTD interest rate and DTI

#### Loan Quality Analysis
- Good loan percentage and applications
- Bad loan percentage and applications
- Funded and collected amount by loan quality

#### Loan Status Summary
- Loan count by status
- Funded amount, amount received
- Average interest rate and DTI by loan status

### B. Bank Loan Report | Overview

#### Month-Wise Analysis
- Loan applications, funding, and collections by month

#### State-Wise Analysis
- Loan distribution across states

#### Term-Wise Analysis
- Loan performance by term

#### Employee Length-Wise Analysis
- Loan trends by employment length

#### Purpose-Wise Analysis
- Loan distribution by purpose

#### Home Ownership-Wise Analysis
- Loan performance by home ownership

## Key Insights

- December recorded the highest funded amount
- Majority of loans are classified as good loans
- Debt consolidation is the most common loan purpose
- Employment length and home ownership impact loan performance
